# 游戏指南

## 程序参数说明

### 设置坦克速度

```shell
# 慢
-DefaultTankSpeed Slow
# 普通
-DefaultTankSpeed Normal
# 快
-DefaultTankSpeed Fast
```

### 设置子弹速度

```shell
# 慢
-DefaultBulletSpeed Slow
# 普通
-DefaultBulletSpeed Normal
# 快
-DefaultBulletSpeed Fast
```
